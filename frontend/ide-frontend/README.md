# IDE Frontend

Frontend application for the Feeder Digital Twin & Design IDE.

## Features

- Topology canvas with drag-and-drop
- Asset palette
- Properties panel
- React Flow integration

## Development

### Prerequisites

- Node.js 20+
- npm

### Run locally

```bash
npm install
npm run dev
```

Application will start on `http://localhost:3000`

### Build

```bash
npm run build
```

### Lint

```bash
npm run lint
```
